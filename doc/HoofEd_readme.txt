HoofEd

Overview
	HoofEd è uno strumento che aiuta la redazione dei contenuti nel CMS.
	HoofEd, pensato per essere leggero e non invasivo,Sviluppato utilizzando 
	il framework MooTools.
	In coppia con la textarea standard consente di inserire markup facilmente.
	Si presta ad essere uno strume
	nto per utenti esperti in quanto attualmente 
	non effettua nessun controllo sul markup generato. 
	
	Ci di un file di configurazione e i file delle lingue.
	Nel file di configurazioni vanno dichiarati tutti i tag disponibili
	Nei file delle lingue vengono tradotte le etichette per i bottoni dell'editor.

Funzionalità
	tag configurabili
	numero di bottoni configurabili
	etichette dei bottoni multilingua
	crossbrowser
	multi-istanza
	possibilità di modificare i bottoni dinamicamente
